<태그 속성 = "값">
속성 - 태그마다 여러속성을 부여해야할 경우
    * 미리 정의된 속성
    * 특별한 기능을하는속성

값 - 속성이 가지는 값
    * 큰따옴표("") 작은따옴표('')
    * 속성이 가진 고유 값
    * 개발자가 기본 규칙 내에서 지정할 수있는 사용자지정 값 설정

--------------리눅스 명령어 기초연습--------------
1. ~ 가장 최상위 디렉터리(폴더) 위치 확인하기
2. ctrl + j -> git bash 터미널 열기
3. pwd 현재경로 디렉터리 표시
4. ls 현재 디렉터리 폴더와 파일 표시
5. mkdir (생성디렉터리명)
    현재경로위치에 디렉터리 생성
6. rm -r 삭제디렉터리명
    현재경로위치에 디렉터리 제거(내용까지 모두)
7. cd 디렉터리명
    하위 디렉터리로 이동
8. cd .. 상위 디렉터리는 이름을 인식할 수 없으므로 한단계 위(..)라는 뜻으로 ..을 사용해서 나간다.
9. vim 생성파일명 .확장자
    원하는 파일을 확장자 포함해서 ex. test.txt 생성후
    i 를 눌러서 끼워넣기 버전으로 변경하고 
    원하는 내용 입력 후 esc 키 누르고 :wq 저장하고 종료
10. cat 확인파일명 .확장자
    내용 확인하고 싶은 파일명 확장자 포함 ex. test.txt 작성 후 확인하면 내용을 미리 볼수 있다.
